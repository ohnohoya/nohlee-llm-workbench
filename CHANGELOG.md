# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-06
### Highlights
- Python runner for OpenAI model experiments with configurable model, endpoint, temperature, and reasoning settings.
- FastAPI backend for running batches and loading saved results.
- React/Vite UI for submitting runs and comparing outputs.
- Docker support to run API + UI together with one command.
- Beginner/Advanced documentation and project setup guides.
- Open-source release basics: MIT license and release changelog.
